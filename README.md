# YSI ProQuatro Multiparameter Meter Simulator

An interactive web-based simulator of the YSI ProQuatro handheld water quality meter, built for the Toolkit for the Field Biologist virtual field trip.

## Features

- Power on/off with splash screen
- Backlight toggle
- Live fluctuating sensor readings (DO, pH, SpCond, Temp, ORP)
- Full menu navigation: System, Sensor, Calibration, File
- Calibration workflow with stabilisation progress bar
- Data logging

## Embed

```html
<iframe
  src="https://yourusername.github.io/proquatro-simulator/"
  width="400"
  height="620"
  frameborder="0"
  scrolling="no"
  style="border:none;">
</iframe>
```

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| P | Power on/off |
| B | Backlight |
| Arrow keys | Navigate |
| Enter | Select |
| Esc | Back |
| 1–4 | Menu keys (System, Sensor, Cal, File) |

## Built with

HTML / CSS / JavaScript — no dependencies, fully self-contained.
